# **App Name**: Sarkari Mitra

## Core Features:

- Scheme Explanation: Uses an AI tool to provide simple, clear explanations of government schemes in multiple languages based on user queries.
- Multilingual Support: Offers support for multiple Indian languages including Hindi, Telugu, Tamil, and Gujarati using a translation API.
- Eligibility Check: Asks a series of simple yes/no questions to determine a user's eligibility for a particular scheme, using an AI tool to decide whether or not the eligibility criteria are met based on a decision flow.
- Application Links: Provides official links to apply for the schemes, ensuring users can easily access the application process.
- WhatsApp Integration: Integrates with WhatsApp via Twilio or Meta WhatsApp Business API to deliver information directly to users.
- User Input Processing: Processes user input from WhatsApp to detect the scheme name and query.
- Clear Output Display: Presents the scheme explanation, eligibility results, and official links in a clear and easy-to-understand format within WhatsApp.

## Style Guidelines:

- Primary color: Calm blue (#7EC8E3) for trustworthiness and clarity, reflecting government services.
- Background color: Light, desaturated blue (#E5F3F7), for a clean, approachable interface.
- Accent color: Warm yellow (#F4D35E) to highlight important links and action items, analogous to the primary color for a cohesive feel.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and readable design.
- Note: currently only Google Fonts are supported.
- Use simple, clear icons to represent different schemes and categories, ensuring easy recognition and understanding.
- Maintain a clean and structured layout, with clear sections for the scheme explanation, eligibility check, and application links.
